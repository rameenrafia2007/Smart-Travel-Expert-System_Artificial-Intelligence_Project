# Smart-Travel-Expert-System_Artificial-Intelligence_Project
A rule-based **Artificial Intelligence Expert System** developed in **SWI-Prolog**  that helps users plan trips intelligently by providing personalized travel  recommendations based on their preferences and budget.

## 📌 About the Project

This project was developed as part of the **Artificial Intelligence Lab** 

The system uses a **knowledge base** with facts and rules stored in Prolog, 
and an **inference engine** that matches user preferences to produce smart 
travel suggestions — mimicking the decision-making of a human travel expert.

---
## Tool & Technology Used

- **Language:** SWI-Prolog
- **AI Technique:** Rule-Based Expert System / Knowledge-Based Reasoning
- **Interface:** Interactive CLI (Chatbot-style menu)

---
## Features

| Feature | Description |
|---|---|
| 🗺️ Trip Recommendation | Suggests destinations based on travel type & budget |
| 🛂 Visa Check | Checks visa requirements for Pakistani travelers |
| 💰 Cost Estimate | Estimates travel cost in PKR |
| 🛡️ Safety Advisory | Lists safe travel destinations |
| 🍜 Famous Food | Suggests famous cuisine of a destination |
| ⭐ Best Rated Destinations | Shows top-rated destinations (4.5+) |
| 📞 Emergency Contacts | Provides emergency contact info per destination |
| 🗓️ Itinerary Planning | Generates day-wise activity plans |

## 🚀 How to Run

1. Install **SWI-Prolog** from [https://www.swi-prolog.org](https://www.swi-prolog.org)
2. Clone this repository:
git clone https://github.com/your-username/smart-travel-expert-system.git
3. Open SWI-Prolog and load the file:
```prolog
   ?- consult('travel_management.pl').
```
4. Start the system:
```prolog
   ?- start.
```
5. Complete Project
